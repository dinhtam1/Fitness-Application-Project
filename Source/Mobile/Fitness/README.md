## Installing

1. Project

Clone the repository

```bash
git clone https://github.com/dinhtam1/Fitness-Application-Project.git

```

Run `npm install` in the project.

```bash
npm install
```

## Usage

- Run the server in development mode

1. Open project in vscode

2. Run the server

```bash
npm run start
```

3. Change config URL

Go to app > config > env.js
